# Projeto de Análise de Dados do Instagram

## Descrição
Realizei uma análise exploratória de dados de postagens do Instagram para identificar padrões de engajamento.

## Ferramentas utilizadas
- Python
- Pandas
- Matplotlib

## Análises realizadas
- Tratamento de valores nulos
- Identificação dos posts com mais curtidas e comentários
- Análise de engajamento por tipo de conteúdo (Foto, Vídeo, Reels)
- Visualização de dados com gráficos
